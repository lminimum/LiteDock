## 技术栈

- **Vue 3** - Composition API + `<script setup>`
- **Vue Router 4** - 现代化路由
- **Pinia** - 状态管理
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Axios** - HTTP 客户端
